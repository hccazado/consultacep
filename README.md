# Overview

## Consulta CEP

A java app that asks for a CEP (Brazil's postal code), and retrieves information about this postal code.
This is a terminal app intended for building skills constructing classes, exceptions, lists and http requests.

## Development Environment

- Chosen IDE: IntelliJ Community Edition
- Java 25
- Java is a versatile object-oriented programming language. One of its key features is the ability to write once, run anywhere because compiled code runs on the Java Virtual Machine (JVM).

## Dependencies

- Gson - A library used to convert JSON into Java objects and vice versa.
